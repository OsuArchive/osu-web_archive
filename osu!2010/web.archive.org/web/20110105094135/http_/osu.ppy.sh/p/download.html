System Requirements

  • 1GHz+ Processor
  • 512mb+ RAM
  • Microsoft .NET Framework (2.0+, 32bit)

Remember:

  • If you have problems that can't be solved by reading the FAQ or searching the forums, please voice them in the forums or on IRC. There will likely be someone that can help you out.
  • If you get an initialisation error when running osume.exe, you may need to manually install the Microsoft .NET Framework.
