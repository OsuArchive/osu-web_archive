System Requirements:
  • Recommended 1ghz+ CPU (2ghz+ for smooth video)
  • Minimum 256mb RAM (512mb+ recommended)
  • DirectX9 and a graphics card that supports DirectX9 (64mb+ of video ram recommended)
  • Microsoft .NET Framework 2.0
  • FFDShow or equivelant video codecs (only for video support)

Instructions:

  1. Download osume and save to a new folder.
  2. Run osume.exe and let it update. This will populate the directory it's in with subfolders and related files so make sure you put it in it's own folder if you haven't already.
  3. Download some beatmaps from the beatmap listing (or visit the forum for unofficial submissions) and place in the songs/ directory. IE USERS: The beatmaps will download as zip files - DO NOT EXTRACT these - simply place them in your Songs folder intact! (You need a forum account for this step - registration is fast and simple. Please introduce yourself if you have the time :))
  4. Play! If you are unfamiliar with the gameplay style, please read the guide on the FAQ

Current Features:

  • Daily updates (and weekly public releases)!
  • Gameplay/scoring feels almost identical to the original.
  • Complete editor featuring easy to use tools.
  • Video and image support for backgrounds.
  • Ever-growing selection of songs to play.
  • Online ranking keeps gameplay fresh as you compete for your place on the charts.
  • Replay mode lets you watch your own high scores or those of competitors online!
  • Full in-game universal and per-beatmap skinning support.

Remember:

  • This game is still in beta. Don't expect it to run perfectly on every system!
  • That said, if you DO have problems - please voice them in here! Feedback is the best way to keep things improving :).
  • If you get an initialisation error when running osume.exe, install the .NET Framework using the link above!
